Few-Shot Learning Extraction

Harnessing LLM's power for training free extraction!

/ The Background

Minimising effort in training AI model by adopting LLM

In the old days before LLM, users will have to label a massive amount of sample data to train the AI model to run extraction. Imagine labelling at least 10+ images on the right.Now with LLM, we can say goodbye to massive effort by potentially avoiding some of the trainings.

/ The Solution

LLM Extraction as the First Class Citizen

Instead of having user to choose which method to use, LLM or the traditional FormX model. We set LLM extraction as default as it required minimal setup effort
